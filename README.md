# Tic Tac Toe Game using Python

This project is a console-based Tic Tac Toe game developed using Python as part of the Python Programming Internship at Hex Software.

## Project Description
The game allows two players to play Tic Tac Toe by taking turns. The board is displayed in the console, and players enter positions from 1 to 9. The program checks for winning conditions and draw scenarios.

## Features
- Two-player game (X and O)
- Input validation
- Win condition checking
- Draw detection
- Clear game instructions

## Technologies Used
- Python (Core concepts)
- Functions
- Lists
- Conditional statements
